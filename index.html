Welcome to jQuery Todo Page.

Created this plugin after taking help and guideline from SitePoint tutorial and Sdaityari Plugin. Implement customization in plugin coding pattern to generate the todo at run time. I have plan to improve it more and add options to integrate third party services, update todo items.

Setup Demo

You have to understanding of bower to run the demo. jQuery Todo demo depend on below mention plugins.

  • Bootstrap 3.3.4
  • jQuery 1.11.1
  • jQuery-ui 1.11.1

Run this code in terminal to setup demo

bower update
